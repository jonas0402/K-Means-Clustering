# K-Means Clustering

Unsupervised learning project applying K-Means clustering to classify unlabeled data points into groups based on centroid proximity.

## Overview

Udemy course project implementing K-Means clustering to group unlabeled observations by iteratively assigning data points to the nearest centroid (determined by mean distance).

## Tech Stack

- **Language:** Python (Jupyter Notebook)
- **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

## Key Concepts

- Unsupervised learning (no target labels)
- Centroid initialization and convergence
- Elbow method for optimal K selection
- Cluster visualization

## Getting Started

```bash
pip install scikit-learn pandas numpy matplotlib seaborn jupyter
```
